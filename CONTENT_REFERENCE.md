## Tone Guidelines

### Certification Language
- Use "Pursuing" not "Certified"
- Use "Alignment in Progress" not "Aligned"
- Use "Targeted" not "Approved"
- Always allow document requests

### Partner Language
- Use "Active partner" with clear scope
- Use "Target / MoU stage" for prospects
- Use "Engagement ongoing" for discussions
- Always note "permission pending" if logo not available

### Contact Language
- Direct and professional
- Always provide multiple contact methods
- Specify purpose in CTAs (partnerships, press, certification)

### Status Pills
- **Green**: Active/Confirmed relationships

### Primary Brand Colors
- **Neon** (#00FFC6): Primary CTAs, key highlights
- **Orange** (#FF6D00): Energy, progress indicators
- **Violet** (#7F5AF0): Premium features, partnerships
- **Dark** (#1A1F36): Backgrounds, depth
